# Codegeist 2020

## Installation

1. Install [git], [node], [npm] \(2.7.5+) and [ngrok].
2. Run `npm install`.

## Development

1. Run `ngrok http 3000` and take note of the proxy's `https://..` base url.
2. Set `localBaseUrl: "https://THE_NGROK_BASE_URL"` in `config.json`
3. Run `npm run start` from the
repository root.
4. Install the app at https://bitbucket.org/sxmyu/workspace/settings/addon-management (make sure you enable development mode)

You can manually install/update/uninstall your add-ons from
`https://bitbucket.org/account/user/USERNAME/addon-management`.

[git]: http://git-scm.com/
[node]: https://nodejs.org/
[npm]: https://github.com/npm/npm#super-easy-install
[ngrok]: https://ngrok.com/
